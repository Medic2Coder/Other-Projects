TEST!How to Install and Use VS Code's Live Server Extension

Installation Instructions

  1. In Visual Studio Code, click the Extensions icon in the left column toolbar.
  2. In the Search bar that appears, type Live Server
  3. Select the Live Server extension. It should be the top choice.
  4. In the page for Live Server that appears, click the green Install button.

Usage Instrictions

  • Visual Studio Code mustmust be open to a folder to use Live Server
  • In VS Code's Explorer, right click (Windows) or Control+click(Mac OS)From the contet menu that appears, Select open with Live Server.
