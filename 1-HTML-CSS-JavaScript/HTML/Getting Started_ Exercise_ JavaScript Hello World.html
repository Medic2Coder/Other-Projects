    Getting Started

    Done: View

    Some background on JavaScript and a JavaScript Hello World exercise. 

    2. Exercise: JavaScript Hello World

    Click here if you wish to open this video in its own window.


    Objective

    • Add a small piece of JavaScript to an HTML page using the script element. 


    Instructions

    • Open your course folder in VS Code and create a new file named javascript-hello-world.html.
    • Build out a standard HTML5 document as usual:
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8" />
    <title>JavaScript Hello World</title>
</head>
<body>

</body>

    • Inside the body element, add a script element with the JavaScript content as follows:

    . . .
    <script>
        alert("Hello World!");
    </script>
. . .

    • Open this document in your browser with Live Server and notice the alert dialog that pops up.
    • If you click OK on the dialog, it goes away, and the page is blank since there was no other content in the HTML body.
    • If you refresh the page, the dialog will pop up again. The JavaScript inside the script element runs every time the page is loaded/reloaded.
    • To stop this behavior, wrap the JavaScript code inside a function declaration as follows, giving the function the name of sayHello:

    . . .
    <script>
        function sayHello() {
            alert("Hello World!");
        }
    </script>
. . .

    • Now if you refresh the page in your browser, the alert dialog does not pop up. That's because JavaScript code that's inside of a function will not execute until it is called. 
    • There are multiple ways to call a function from inside an HTML file. Let's add an HTML button to call it using an onclick event handler, as follows:

    . . .
<body>
    <button type="button" onclick="sayHello()">Click Me</button>
    <script>
. . .

    • In your browser, you should now see a button. Click it to see the alert dialog pop up. 
    • Every time you click the button, the function will be called and the alert dialog will pop up again, without needing to refresh the page.
    • Explanations of these concepts (functions, event handlers, etc) will be provided in following lessons. This exercise is meant to give you a taste of writing a little JavaScript and running it from a webpage right away.
